# Lavaux Vineyard Walk — Grandvaux to St-Saphorin

A single-page, static interactive map of a self-guided Sunday walking tour through
the Lavaux vineyard terraces above Lake Geneva.

- **Stack:** plain HTML / CSS / JS — no build step.
- **Map:** Leaflet.js via CDN, CARTO Voyager tiles. No API keys.
- **Features:** 7 stops (walk + wine), toggle layers, click-to-detail panel with
  inline SVG scenes, light/dark mode, responsive, accessible stop list.

## Files
```
index.html    markup + hero + map + detail + itinerary
styles.css    warm editorial theme (light/dark via prefers-color-scheme)
script.js     data, SVG scenes, Leaflet map, toggles, detail panel
```

## Local preview
No build needed. Because tiles load over the network, use a local server rather
than opening the file directly:

```bash
# Python 3
python3 -m http.server 8000
# then open http://localhost:8000
```

Or with Node: `npx serve` (opens on http://localhost:3000).

## Deploy
Any static host works — the site is just three files.

- **GitHub Pages:** push these files to a repo, then Settings → Pages → deploy
  from the `main` branch, root folder.
- **Netlify / Vercel:** drag the folder into the dashboard, or connect the repo.
  No build command, no output directory (publish the root).
- **Render:** create a **Static Site**, connect the repo, leave build command
  empty, set the publish directory to `.` (root).

## Swapping in real photos later
Each stop's illustration is generated by `sceneSVG(kind)` in `script.js`.
To use real photos instead, replace that function's return value with an
`<img src="…" alt="…">` per stop (keep the same container class so styling holds).
